# vue-cli 

### 1.安装vue-cli
```sh
npm install -g vue-cli
```
<!-- more -->
### 2.使用vue-cli初始化项目
```sh
vue init webpack-simple my-project-name
```
### 3、进到目录
```sh
cd my-project
```
### 4、安装依赖
```sh
npm install
```
### 5、开始运行
```sh
npm run dev
```
### 6、测试项目--访问
localhost:8080
